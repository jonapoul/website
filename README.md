## Running locally

```sh
# Clone the repo
git clone https://github.com/jonapoul/jonpoulton.dev
cd jonpoulton.dev

npm install --force # this stage takes a whie, so go grab a cuppa
npm start # should launch the browser to http://localhost:3000
```
